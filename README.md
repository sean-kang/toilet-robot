# Toilet Robot
Toilet Robot that detects motion and sends a notification through wifi.

## Requirements
- [WeMos D1R2](http://www.wemos.cc/Products/d1_r2.html)
- PIR Sensor
- [Arduino 1.6.9](https://www.arduino.cc/)
- [ESP8266 Core](https://github.com/esp8266/Arduino)
